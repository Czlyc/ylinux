ylinux 项目的目录结构：

使用 Django 开发，项目名称为 ylinux 。目前项目源码存放在 github 上。其中 "lib/" 目录可存放一些第三方软件，建议把 django 项目最新源代码也放到这里，这样在服务器上部署实际的项目时，可以不受限于服务器的 django 版本，其他需要用到的纯 Python 库，也建议存放在这个目录。

ylinux/
  - app/                    为ylinux项目开发的应用
      - home/
      - account/
      - ydata/
      - forum/

  - lib/                    第三方python库，包括 django

  - templates/              所有模板
      - home/               home 应用的模板

  - ymedia/                 css,js,images
